The Business Dashboard is a data visualization tool that centralizes key metrics and KPIs into a single, clean interface — giving business owners and stakeholders a clear view of performance at a glance.

Developer Note: This project was built to demonstrate my proficiency in modern Angular development. It reflects my approach to frontend engineering: clean architecture, component-based design, and intuitive user experiences. The app uses static/mock data to keep the focus on UI structure, performance, and maintainability.

Built with Angular, TypeScript, SCSS, and Angular Material — backed by a lightweight PHP & MySQL backend.
